## Overview

The Meridian Pulse Button is a premium interactive action component designed for modern interfaces that require strong visual emphasis, elegant motion design, and immersive user feedback.

Featuring a luminous gradient surface, floating particle effects, animated vector iconography, and a dynamic corner accent interaction, the component delivers a sophisticated experience while preserving lightweight implementation and framework portability.

The design is engineered to create a sense of energy, motion, and responsiveness, making it particularly effective for high-priority actions, onboarding flows, platform credits, account upgrades, AI products, and modern SaaS applications.

## Interaction States

### Default

Displays the primary gradient surface with floating particles and elevated depth.

### Hover

* Corner accent retracts outward
* Shadow elevation increases
* Particle field remains active
* Icon animation sequence begins
* Component gains additional visual prominence

### Active

* Subtle scale compression for tactile feedback
* Preserves animation continuity

---

## Visual Effects

### Floating Particle System

Ten independently animated particles continuously travel upward through the component surface, creating a premium kinetic effect.

### Corner Accent

A folded-corner inspired visual detail adds depth and character while providing an elegant hover transition.

### Vector Animation

The lightning icon utilizes animated stroke rendering and delayed fill transitions to create a refined interaction experience.

### Gradient Surface

Multi-layer radial and linear gradients establish a high-end visual hierarchy while maintaining clarity and readability.
