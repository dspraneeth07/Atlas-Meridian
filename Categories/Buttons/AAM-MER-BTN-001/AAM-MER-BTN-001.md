## Overview

The Meridian Launch Button is a premium call-to-action component designed for modern web interfaces requiring strong visual hierarchy and polished interaction feedback.

Built with a refined gradient surface, animated icon expansion, elevated shadow system, and smooth micro-interactions, the component provides a high-end user experience while maintaining a clean and adaptable structure.

The design emphasizes clarity, responsiveness, and visual sophistication, making it suitable for landing pages, dashboards, SaaS products, portfolios, developer tools, and enterprise applications.

## Interaction States

### Default

Displays the primary gradient surface with elevated depth and icon container alignment.

### Hover

* Icon container expands horizontally
* Arrow icon shifts forward
* Shadow elevation increases
* Highlight sweep animation becomes visible

### Active

* Component scales subtly for tactile feedback
* Icon container compresses slightly
